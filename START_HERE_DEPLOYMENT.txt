STEP 2: MONGODB ATLAS (Database)
─────────────────────────────────

1. Sign up: https://mongodb.com/cloud/atlas/register
   - Use Google/GitHub to sign up

2. Create FREE cluster:
   - Choose: M0 Free
   - Region: Closest to you
   - Name: car-service

3. Create database user:
   - Username: carservice
   - Password: Auto-generate (SAVE IT!)

4. Network access:
   - Add IP: 0.0.0.0/0 (allow all)

5. Get connection string:
   - Click "Connect" → "Connect your application"
   - Copy URL:
     mongodb+srv://carservice:PASSWORD@cluster.mongodb.net/carservice

✅ Database URL saved!

═══════════════════════════════════════════════════════════════════

STEP 3: RENDER (Backend)
────────────────────────

1. Sign up: https://render.com (with GitHub)

2. New Web Service:
   - Connect: car-service-mern repo
   - Name: car-service-api
   - Root Directory: backend
   - Build: npm install
   - Start: node src/server.js
   - Type: FREE

3. Environment Variables (Click Advanced):
   
   MONGODB_URI = [Your Atlas connection string]
   JWT_SECRET = create_random_secret_here
   EMAIL_USER = manpreet123singh987@gmail.com
   EMAIL_PASSWORD = plsohynfxgbdkapo
   EMAIL_SERVICE = gmail
   EMAIL_FROM_NAME = Car Service Platform
   NODE_ENV = production
   FRONTEND_URL = http://localhost:3000

4. Click "Create Web Service"

5. Wait 5-10 minutes...

6. Get your URL:
   https://car-service-api.onrender.com

✅ Backend API is live!

═══════════════════════════════════════════════════════════════════

STEP 4: VERCEL (Frontend)
──────────────────────────

1. Sign up: https://vercel.com (with GitHub)

2. Import Project:
   - Select: car-service-mern
   - Root Directory: frontend
   - Framework: Create React App

3. Environment Variable:
   
   REACT_APP_API_URL = https://car-service-api.onrender.com

4. Click "Deploy"

5. Wait 3-5 minutes...

6. Get your URL:
   https://car-service-mern.vercel.app

✅ Frontend is live!

───────────────────────────

3. Update FRONTEND_URL:
   From: http://localhost:3000
   To: https://car-service-mern.vercel.app
